# KI_1

Assignment 1 in IT3105 - Artificial Intelligence programming course at NTNU.
The code implements a reinforcement learning environment representing the one player board game Peg Soltaire. Also an Actor-Critic reinforcement learning model is implemented, which is able to solve the board game, only by observing the states, legal actions and recieving rewards

# Usage

The jupyter notebook "Demo.ipynb", demonstrates the use of the modules.
